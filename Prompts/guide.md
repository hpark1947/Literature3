# 논어 AI 분석 프롬프트 가이드

논어를 AI로 분석할 때 활용할 수 있는 프롬프트 작성 요령과 템플릿을 정리한 가이드입니다.

---

## 1. 기본 원칙

### 맥락을 먼저 제공하라
AI에게 논어의 배경 지식을 함께 전달하면 분석 품질이 높아진다.

```
논어(論語)는 공자(BC 551~479)와 제자들의 대화를 기록한 유교 경전으로,
총 20편 482장으로 구성되어 있다.
아래 구절을 이 맥락에서 분석해 주세요.
```

### 분석 관점을 명시하라
"분석해 줘"보다 구체적인 관점을 지정한다.

| 관점 | 질문 예시 |
|------|-----------|
| 개념 분석 | "이 구절에서 인(仁)은 어떤 의미로 쓰였는가?" |
| 인물 분석 | "공자는 왜 안연과 자로에게 다른 답을 했는가?" |
| 구조 분석 | "학이편의 16장은 어떤 논리적 흐름으로 배열되었는가?" |
| 비교 분석 | "인(仁)에 대한 안연편과 옹야편의 정의를 비교하라" |
| 현대 적용 | "이 구절을 현대 리더십에 어떻게 적용할 수 있는가?" |

### 원문을 함께 제공하라
한자 원문 + 독음 + 기존 해석을 함께 주면 AI가 더 정확하게 분석한다.

```
원문: 子曰 學而時習之 不亦說乎
독음: 자왈 학이시습지 불역열호
기존 해석: 배우고 때때로 익히면 기쁘지 아니한가
→ 여기서 '習(습)'의 의미를 '반복 학습'과 '실천' 두 관점에서 분석해 주세요.
```

## 2. 용도별 프롬프트 템플릿

### 2.1 구절 심층 해석

```
다음 논어 구절을 심층 분석해 주세요.

[원문]: (한자)
[독음]: (한글)
[출처]: (편명 + 장번호)
[기존 해석]: (통용되는 번역)

분석 요청:
- 핵심 글자의 의미 변천 (고대 → 현대)
- 문장 구조와 수사법
- 이 구절이 놓인 편 전체에서의 맥락
- 다른 구절과의 연결 관계
- 주희(朱熹) 등 주요 주석가의 해석 차이 (알고 있다면)
```

### 2.2 개념 비교 분석

```
논어에서 [개념 A]와 [개념 B]를 비교 분석해 주세요.

[개념 A]: (예: 인/仁)
[개념 B]: (예: 예/禮)

분석 항목:
- 각 개념의 정의와 범위
- 두 개념의 관계 (상호보완 / 상하 / 대립)
- 관련 구절 대조
- 공자 사상 체계에서 각 개념의 위치
```

### 2.3 인물 분석

```
논어에 등장하는 [인물]을 분석해 주세요.

[인물]: (예: 안연/顔淵, 자로/子路, 자공/子貢)

분석 항목:
- 공자와의 관계 및 공자의 평가
- 해당 인물의 질문 패턴과 관심사
- 공자가 이 인물에게만 한 가르침의 특징
- 다른 제자와의 비교
- 인물이 체현하는 유교적 덕목
```

### 2.4 편(篇) 단위 구조 분석

```
논어 [편명]편 전체의 구조를 분석해 주세요.

[편명]: (예: 학이/學而)
[장 수]: (예: 16장)

분석 항목:
- 편의 전체 주제와 핵심 메시지
- 장(章)별 주제 흐름과 배열 논리
- 편 내에서 반복되는 핵심 키워드
- 다른 편과의 관계
- 이 편이 논어 전체에서 차지하는 위치
```

### 2.5 현대 적용 분석

```
다음 논어 구절의 현대적 의미를 분석해 주세요.

[구절]: (원문 + 해석)
[적용 분야]: (택 1)
  - 리더십/경영
  - 교육
  - 인간관계/소통
  - 자기 수양/멘탈 관리
  - 정치/사회

분석 항목:
- 원래 맥락에서의 의미
- 현대 상황에 대입했을 때의 해석
- 구체적 적용 사례나 시나리오
- 한계점이나 주의할 점
```

### 2.6 사자성어 유래 분석

```
다음 사자성어의 논어 속 유래를 분석해 주세요.

[사자성어]: (예: 溫故知新)
[한글]: (예: 온고지신)
[일상적 의미]: (예: 옛것을 익혀 새것을 안다)

분석 항목:
- 논어 원문에서의 정확한 출처와 전후 맥락
- 원래 의미와 현재 통용되는 의미의 차이
- 관련된 다른 논어 구절
- 한국 문화에서의 활용 사례
```

### 2.7 주제별 구절 수집

```
논어에서 [주제]에 관한 구절을 모아 분석해 주세요.

[주제]: (예: 배움의 자세, 친구 관계, 말과 행동의 일치)

요청 사항:
- 해당 주제와 직접 관련된 구절 수집 (원문 + 해석)
- 구절 간의 공통점과 차이점
- 구절들을 종합했을 때 드러나는 공자의 일관된 관점
- 주제에 대한 종합 요약
```

## 3. 분석 깊이 조절

상황에 따라 분석 깊이를 지정한다.

| 수준 | 지시 문구 | 용도 |
|------|-----------|------|
| 간략 | "핵심만 3줄로 요약해 주세요" | 빠른 참고 |
| 보통 | "주요 포인트를 항목별로 정리해 주세요" | 일반 학습 |
| 심층 | "학술적 수준으로 상세히 분석해 주세요" | 깊은 연구 |
| 비교 | "A와 B를 표로 대조하여 분석해 주세요" | 개념 비교 |

## 4. 프롬프트 개선 기법

### 단계적 분석 (Chain of Thought)
복잡한 분석은 단계를 나누어 요청한다.

```
1단계: 이 구절의 핵심 글자(仁, 禮 등)의 의미를 풀어 주세요
2단계: 문장 전체의 뜻을 해석해 주세요
3단계: 이 구절이 속한 편의 맥락에서 어떤 역할을 하는지 분석해 주세요
4단계: 현대적 시사점을 도출해 주세요
```

### 역할 부여
```
당신은 동양철학 전공 교수입니다.
논어를 30년간 연구해 왔으며, 주희의 집주와 다산 정약용의 논어고금주를
모두 숙지하고 있습니다.
```

### 출력 형식 지정
```
다음 형식으로 정리해 주세요:
- 원문 | 독음 | 해석 (표 형식)
- 핵심 개념 (굵은 글씨로 강조)
- 현대적 의미 (별도 단락)
```

### 제외 사항 명시
```
- 지나치게 현대적인 비유는 피해 주세요
- 원문의 뉘앙스를 왜곡하지 말아 주세요
- 확실하지 않은 해석은 "~로 볼 수 있다"로 표현해 주세요
```

## 5. 자주 쓰는 프롬프트 예시

### 빠른 해석
> "子曰 三人行 必有我師焉"을 해석하고, 현대 직장 생활에 적용할 수 있는 교훈을 알려 주세요.

### 개념 탐구
> 논어에서 '군자(君子)'가 등장하는 구절을 5개 이상 모아서, 공자가 생각한 군자의 조건을 종합해 주세요.

### 비교 분석
> 공자의 "기소불욕 물시어인(己所不欲 勿施於人)"과 기독교의 황금률("남에게 대접받고자 하는 대로 남을 대접하라")을 비교 분석해 주세요.

### 구조 파악
> 논어 위정편(爲政篇)의 24장을 주제별로 묶어 정리하고, 편 전체의 논리적 흐름을 설명해 주세요.

### 현대 적용
> "학이불사즉망 사이불학즉태(學而不思則罔 思而不學則殆)"를 AI 시대의 교육에 적용하여 해석해 주세요.
