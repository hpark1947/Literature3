# Prompts

문학 창작 및 분석을 위한 AI 프롬프트 가이드와 템플릿 모음입니다.

---

## 폴더 구조

```
Prompts/
├── README.md          ← 현재 문서
├── guide.md           ← 프롬프트 작성 가이드
├── novel.md           ← 소설 창작 템플릿
├── poetry.md          ← 시 창작 템플릿
├── analysis.md        ← 문학 분석 템플릿
├── editing.md         ← 퇴고 및 피드백 템플릿
├── worldbuilding.md   ← 세계관 구축 템플릿
└── character.md       ← 인물 설계 템플릿
```

---

## 파일 설명

### guide.md — 프롬프트 작성 가이드
AI에게 문학 작업을 요청할 때의 기본 원칙과 작성 요령을 정리한 문서입니다.
- 명확성, 맥락 제공, 제약 조건 설정 등 기본 원칙
- 단계적 요청, 역할 부여, 예시 제공 등 개선 기법
- 자주 하는 실수와 활용 팁

> 처음 사용한다면 이 문서를 먼저 읽어 주세요.

### novel.md — 소설 창작
장르, 배경, 인물, 서사 구조, 문체를 설정하여 소설을 요청하는 템플릿입니다.
- 인물 설정 (주인공 + 주요 인물 2명)
- 5단계 서사 구조 (발단 → 갈등 → 위기 → 절정 → 결말)
- 참고 스타일 및 제외 사항

### poetry.md — 시 창작
주제, 형식, 이미지, 감각 요소를 설정하여 시를 요청하는 템플릿입니다.
- 오감 기반 이미지 설계
- 연/행 구성 및 리듬 설정
- 참고 시인 및 스타일 지정

### analysis.md — 문학 분석
특정 작품에 대한 분석을 요청하는 템플릿입니다.
- 8가지 분석 관점 (서사 구조, 인물, 주제 의식, 문체, 상징 등)
- 분석 깊이 및 분량 조절
- 출력 형식 선택 (에세이 / 항목별 / 표)

### editing.md — 퇴고 및 피드백
완성된 원고에 대한 검토와 수정 제안을 요청하는 템플릿입니다.
- 3단계 검토 (문장 → 장면 → 구조)
- 피드백 방식 및 톤 설정
- 작가 의도 보존 항목

### worldbuilding.md — 세계관 구축
작품 배경이 되는 세계를 설계하는 템플릿입니다.
- 핵심 규칙 및 제한 사항
- 사회 구조 (정치, 경제, 문화)
- 지리, 환경, 역사
- 다양한 출력 형식 (설정집 / 연표 / 용어 사전 등)

### character.md — 인물 설계
입체적인 인물을 설계하는 템플릿입니다.
- 외면 (외형, 복장, 버릇)
- 내면 (성격, 가치관, 욕망, 두려움)
- 변화 호 (시작 → 전환점 → 도착)
- 목소리 (말투, 상황별 대사 예시)

---

## 사용법

1. 목적에 맞는 템플릿 파일을 연다
2. 빈칸을 채운다 (모든 항목을 채울 필요는 없음)
3. 작성한 내용을 AI에게 전달한다
4. 결과물을 확인하고 `editing.md`로 퇴고를 진행한다

### 추천 워크플로우

```
character.md → worldbuilding.md → novel.md → editing.md
(인물 설계)    (세계관 구축)      (본문 작성)  (퇴고)
```

```
poetry.md → editing.md
(시 창작)    (퇴고)
```

```
analysis.md
(독립적으로 사용)
```
