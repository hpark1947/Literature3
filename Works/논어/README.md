## 논어 한눈에 보기

- **정의**: 공자(BC 551~479)와 제자들의 대화를 기록한 유교 핵심 경전
- **구성**: 총 20편, 482장, 약 600여 문장
- **편찬**: 공자 사후 제자들이 수 차례에 걸쳐 편찬
- **지위**: 사서(四書: 논어·맹자·대학·중용)의 으뜸

|------|------|------|-----------|
| 인 | 仁 | 사람다움, 사랑 | 克己復禮爲仁 |

| 예 | 禮 | 사회 규범, 질서 | 人而不仁 如禮何 |
| 지 | 智 | 참된 앎, 지혜 | 知之爲知之 |
| 효 | 孝 | 부모 공경 | 孝弟也者 其爲仁之本 |
| 충/서 | 忠/恕 | 성실함/헤아림 | 己所不欲 勿施於人 |
| 군자 | 君子 | 이상적 인간상 | 君子和而不同 |
